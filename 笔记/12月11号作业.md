# 项目二：猜测数字验证码

功能：

1. 询问是否开始游戏（Y/N）

2. 生成随机四位数字验证码（无重复数字 1-9）

3. 用户一次性输入四位数字验证码

4. 输入验证码（3）与生成的验证码（2）进行比较，显示结果（数字和位置都对为A 数字对位置不对为B）

5. 如果猜中数字验证码，显示OK，显示猜测次数

6. 执行1操作

 

例子：系统生成验证码为6583；

输入 4537  显示1A1B

输入 4589  显示2A0B

输入 8356  显示0A4B

输入 1279  显示0A0B
